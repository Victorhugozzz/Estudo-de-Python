# 🐍 Estudo de Python

Este repositório tem como objetivo documentar minha evolução e aprendizado em Python.  
Aqui estão reunidos todos os desafios propostos pelo curso de Python do Gustavo Guanabara, organizados de forma prática, com soluções otimizadas, explicações e aplicação de boas práticas de programação.

> 📚 Todos os créditos dos desafios pertencem integralmente ao professor Gustavo Guanabara e ao projeto Curso em Vídeo.
